# 初学者学习建议与计划

## 6.1 学习路径规划

### 第一阶段：理解基础概念

- 学习药品临床综合评价的定义、目的、适用范围
- 理解基本原则与核心维度

### 第二阶段：掌握评价流程

- 熟悉主题遴选的方法与标准
- 学习评价实施的步骤与关键环节
- 掌握结果转化的应用路径

### 第三阶段：深入维度分析

- 安全性评价方法
- 有效性证据分析
- 经济性评价技术（如成本-效果分析、成本-效用分析）
- 创新性与适宜性评估
- 可及性评价指标

### 第四阶段：实践与案例学习

- 参与模拟评价项目
- 学习真实世界数据分析
- 阅读已有的药品临床综合评价报告

### 第五阶段：参与质量控制与成果转化

- 学习质量控制要点
- 理解结果应用机制
- 熟悉数据平台与信息化工具

## 6.2 学习资源推荐

### 官方指南

国家卫生健康委发布的《药品临床综合评价管理指南（2021年试行）》

### 相关法规

如《基本医疗卫生与健康促进法》《药品管理法》等

### 学术平台

PubMed、CNKI、万方等数据库

### 培训课程

国家级药品综合评价培训项目、在线慕课等
